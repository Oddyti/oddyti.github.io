Windows软件分享/新系统配置清单

最近给工位电脑配置的了新的工作环境，所以记录一下新工作环境需要装的软件。

2023年10月31日
阅读耗时: 2 分钟

  • 浏览器 - Edge

    Win11自带。

  • Office全家桶

    无须解释。

  • QQ

    无须解释。

  • 微信

    无须解释。

  • 卸载软件 - Geek Uninstaller - the best FREE uninstaller

    免费，一个非常小巧的卸载软件的软件，可以扫描到系统中几乎所有安装的软件，以及卸载后扫描清理注册表功能。

  • 视频播放 - Potplayer

    免费，功能强大的播放器。建议换一个皮肤，默认皮肤太丑了。

  • 写作 - Typora

    收费，好用的Markdown编辑器。搭配自己根据Drake修改的皮肤。

  • 工作 - GitHub Desktop

    免费，Github仓库管理及Git管理，自从用了它，再也没有手敲过git指令。

  • 娱乐 - 网易云音乐

    免费，听歌。

  • 工作 - Zotero

    免费，论文管理+阅读，支持翻译插件（推荐zotero-pdf-translate），云同步（搭配坚果云使用）。

  • 工作 - Visual Studio Code

    免费，最好用的编辑器。

  • 压缩 - Bandizip

    免费，解压缩软件，轻量。

  • 图像查看 - Honeyview

    免费，界面丑，但是好用，支持自定义鼠标和键盘操作。（如滚轮缩放）

  • 工作 - Anaconda

    免费，Python虚拟环境、包管理。装好记得配置系统环境变量。

  • 工作 - MATLAB

    收费，无须解释。

  • 工作 - PowerShell 7

    免费，建议升级自带的Powershell 5到Powershell 7。配置PSReadLine插件实现补全。不建议Oh My Posh美化，加载配置文件太慢。

  • 工作 - Git

    免费，无须解释。

  • 截图 - Snipaste

    小巧好用的截图软件。

  • 系统 - EarTrumpet

    可以分软件调整音量的轻量软件。

  • 系统 - Capslock+

    免费，赋予Capslock类似于Shift一样的功能，可以组合不同的快捷键。
